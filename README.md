# Movie-Recommendation-System-Java
A collaborative filtering-based movie recommendation system built in Java using weighted averages and filters.

It mimics the behavior of platforms like Netflix or Amazon Prime by providing movie recommendations based on user ratings and preferences.

## 💡 Features
- Calculates average movie ratings
- Filters movies based on genre, year, and other criteria
- Uses collaborative filtering to recommend movies from similar users
- Weighted rating system for more accurate recommendations

## 📁 Project Structure
- `StepOneStarterProgram`: Calculate average movie ratings
- `StepTwo-SimpleRecommendations`: Improve rating calculation and generate basic recommendations
- `StepThree-Interfaces-Filters-Database`: Add filters and modular structure
- `StepFour-WeightedAverages`: Calculate weighted ratings using similar raters
- `StepFive-Final`: Final complete recommendation system

## 🛠️ Technologies Used
- Java
- CSV File handling
- Object-Oriented Programming
- Simple text-based user interface

## 📸 Sample Output
> Recommendations are sorted by weighted average ratings from users with similar preferences.

## 🧠 Recommendation Logic
- Finds raters most similar to you
- Weights their ratings based on similarity
- Ranks movies based on weighted average scores
